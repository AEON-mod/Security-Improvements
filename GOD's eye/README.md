👁️ GOD's eye: Advanced System Diagnostics
A collection of elite-level diagnostic utilities from Microsoft Sysinternals to monitor, analyze, and control everything happening under the hood of Windows.

📁 Tools Included
| Tool | Purpose |
|---|---|
| Autoruns | Shows you exactly what programs are configured to run during system bootup or login. It is the most comprehensive autostart monitor available. |
| Process Explorer | A super-powered Task Manager. It shows you which handles and DLLs processes have opened, their CPU/RAM usage, and identifies who owns a specific process. |
| TCPView | Provides a real-time list of all active TCP and UDP connections on your local system, including the remote addresses and state of network connections. |

🚀 How to Use
These tools are "portable," meaning they don't require installation.
 * Open the Folder: Navigate to the specific tool you wish to use.
 * Run as Administrator: Right-click the .exe (e.g., procexp.exe or autoruns.exe) and select Run as Administrator to ensure the tool can see system-level processes.
 * Accept EULA: On the first run, you will be prompted to accept the Microsoft Sysinternals License Agreement.

💡 Pro-Tips:
 * In Process Explorer: Go to Options > VirusTotal.com > Check VirusTotal.com to automatically scan every running process against 70+ antiviruses.
 * In Autoruns: Look for "Pink" or "Yellow" highlighted rows—these often indicate unsigned files or missing paths that could be potential security risks.
 * In TCPView: Right-click any suspicious connection to "Close Connection" or "Whois" the IP address to see where your data is going.

⚠️ Disclaimer
These are advanced utilities. Be careful when "killing" processes or "deleting" autostart entries unless you are certain they are unnecessary or malicious, as removing critical system files can cause OS instability.
